# Ontahí Library

This repository is not a documentation site.

It is the evolving body of knowledge behind Ontahí: its ideas, principles, philosophy, architecture, language, and long-term memory.

Documentation explains how something works.

Books explain why it exists.

Ontahí needs both. This repository begins with the second.

## The First Shelf

The library starts with one book:

- [Living Systems](library/01-living-systems/)

The name is intentional. The first book should not begin by explaining Ontahí. It should begin by explaining systems: how they change, how they preserve identity, how complexity accumulates, and why architecture exists to manage the cost of change.

Ontahí should appear later, as a natural consequence.

## Repository Principles

Treat this repository as a living organism.

The structure may evolve. Books may split. Chapters may move. Ideas may be rewritten. The organization should emerge as understanding grows.

Do not optimize the library too early.

Let the ontology of the writing evolve in the same way Ontahí expects software systems to evolve.
